RSA Implementation

github.com/jmc529/RSA 

This project implements the RSA algorithm, one of the first public-key cryptosystems. The program generates pseudo-random private and public keys, within the limits of an integer ‘n’ of 2^32, and encrypts and decrypts numerical messages. I have recently expanded this project to work in python for a cryptography course I am enrolled in.
